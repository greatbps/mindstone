Git Hub 사용법 #Source Tree 사용법

1. Github 에서 New 프로젝트 생성
2. 먼저 Github에서 Clone할 저장소(Repository)의 주소를 복사
3.  `Clone or Download` 버튼을 클릭합니다. 
4. 클립보드에 복사하기 버튼을 클릭
5. 하단의 커밋을 클릭한다.
6. 상단 Status Bar의 목록 창에서 Push 클릭
